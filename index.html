Heart Disease Detection Using Machine Learning

An ML system that predicts heart disease risk using patient clinical data for early diagnosis and improved healthcare decisions.

Problem Statement

Heart disease remains a major cause of mortality worldwide. Traditional diagnosis methods are slow and prone to human error. This project introduces a machine learning system that analyzes clinical data to provide fast and accurate prediction of heart disease risk.

Literature Review

Existing prediction systems use statistical and rule-based methods with moderate accuracy and limited automation.

Research Gap

There is a need for a scalable ML-based system capable of providing accurate real-time heart disease prediction.

System Methodology

Dataset / Input

The project uses the UCI Heart Disease dataset containing patient clinical attributes such as age, cholesterol, blood pressure, and ECG results. The dataset is preprocessed to remove missing values and normalize features for ML training.

Access Dataset

Model / Architecture

The system applies a supervised machine learning pipeline including preprocessing, feature selection, and classification using Random Forest and Logistic Regression models to predict heart disease risk.

ML Model Reference

ML Training Code 

import pandas as pd
from sklearn.model_selection import train_test_split
from sklearn.ensemble import RandomForestClassifier

data = pd.read_csv("heart.csv")

X = data.drop("target", axis=1)
y = data["target"]

X_train,X_test,y_train,y_test =
train_test_split(X,y,test_size=0.2)

model = RandomForestClassifier()
model.fit(X_train,y_train)

print("Accuracy:",model.score(X_test,y_test))

Results & Analysis

Accuracy 92%

Existing systems achieve approximately 70% accuracy and rely on manual interpretation. The proposed ML solution improves prediction accuracy to 92%, enabling faster and more reliable healthcare decision-making.
